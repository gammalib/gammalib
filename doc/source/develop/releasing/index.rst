Creating a new GammaLib release
===============================

To create a new GammaLib release you have to go through the following steps:

.. toctree::
   :maxdepth: 1

   create
   check
   release
   tag
   merge
   devel


.. important::
   Before releasing the software please make sure that you have **added unit
   tests for all new code** and that you have **updated the User documentation**
   if necessary.
